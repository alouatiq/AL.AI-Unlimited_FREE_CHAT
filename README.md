# FreeChat

A tiny, single-file AI chat that runs on **free OpenRouter models** — no backend, no build step. Open `index.html` and go.

- Automatic **fallback** across free models until one answers
- **Normal / Uncensored** toggle (two model lists)
- **Chat history saved in your browser** (localStorage) — nothing leaves your machine except the API call
- Your OpenRouter key is stored only in the browser

## Use

1. Get a free key at <https://openrouter.ai/keys>.
2. Open `index.html`, paste the key in ⚙ Settings.
3. Chat.

Free models rate-limit and change over time — edit the model lists in Settings anytime.
