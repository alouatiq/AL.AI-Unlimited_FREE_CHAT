# Screenshots

Drop your PNG screenshots here so they render in the main README.

Recommended shots (match the filenames referenced in the root `README.md`):

| File | What to capture |
|------|-----------------|
| `chat.png` | The main chat view with a conversation and the sidebar (groups + chats) |
| `providers.png` | ⚙ Settings → Providers & fallback order + a provider's model picker |
| `ondevice.png` | ⚙ Settings → On-device: downloaded models pinned on top + catalogue |
| `images.png` | Image mode — a generated or edited image in the chat |

Tip: use a ~1440px-wide window, light or dark theme, and crop tightly.
